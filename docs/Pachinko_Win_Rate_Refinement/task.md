# パチンコ勝率計算の高度化

## 現状の課題と目標

ユーザーの要望「とりあえずパチンコ勝率計算もうちょい作り込むか」に基づき、パチンコ・パチスロの収支シミュレーション機能を強化する。
具体的には、単純な独立試行だけでなく、より実践的な指標や複雑なスペック（確変・ST など）への対応、あるいは計算精度の向上を目指す。
